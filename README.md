# TableEat — Prototipo Bootstrap 5

Prototipo funcional desarrollado para la **Práctica 3** de la asignatura *Entorno Web II: Edición y Administración de Contenidos* (CEU 2025-2026).

**Grupo 3 —** María Bermello, Marcela González, Alonso Lara

## Sobre el proyecto

TableEat es una plataforma para encontrar y reservar mesa en restaurantes con disponibilidad de última hora, integrando criterios de estilo, ubicación, presupuesto y tipo de cocina.

## Stack

- **HTML5 + CSS3** (sin preprocesadores)
- **Bootstrap 5.3** vía CDN (sin instalación local)
- **Bootstrap Icons** vía CDN
- **Google Fonts**: Fraunces (display) + Inter (cuerpo)

## Pantallas

| Archivo | Pantalla |
|---|---|
| `index.html` | Landing page con hero, buscador y restaurantes destacados |
| `busqueda.html` | Resultados con filtros laterales (Offcanvas en móvil), grid de cards |
| `restaurante.html` | Ficha con carousel, información detallada y formulario de reserva |
| `confirmacion.html` | Pantalla de confirmación con resumen |
| `styles.css` | Personalización mínima sobre Bootstrap (colores de marca y tipografía) |

## Componentes de Bootstrap utilizados

Navbar con toggler, Grid system (col-lg-*, col-md-*, col-12), Cards, Offcanvas, Carousel, List Group, Badge, Floating Labels, Form controls (range, time, date, select), Breadcrumb, Pagination, Alert, Button group con btn-check.
